## Description

Task for recrutation purpose

## Running the app
Firstly, copy `.env` file:
```bash
cp .env.example .env
```

To run application simple:
```bash
docker compose up
```

or :
```bash
docker-compose up
```

## Documentation

Swagger documentation is available [here](http://localhost:3000/docs)

## Test

To run unit tests
```bash
npm i
npm run test
```
